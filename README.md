# Re0-The-ACM-Journey-Starting-from-Zero
我的 ACM 算法学习笔记仓库，基于 MkDocs + Material 主题构建，风格参考 OIWiki，包含数据结构、算法模板、刷题总结等内容，持续更新中。


## 🚀 项目介绍
- **定位**：ACM 算法学习的个人笔记 + 模板库，聚焦实用代码模板与知识点梳理；
- **风格**：参考 OIWiki 的简洁清爽风格（扁平化、无冗余样式）；
- **技术栈**：MkDocs + Material for MkDocs 主题，支持搜索、代码高亮、数学公式（MathJax）。
